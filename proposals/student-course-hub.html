Student course hub

Proposed by: ggstuart

Description

OK, the idea is that its a website for a university. It will list all the available degree courses. Prospective students can to sign up to a mailing list for more information about e.g. open days. The university get's a mailing list of student emails, with the courses they are interested in. The students get to browse programmes, modules and staff (like module leaders and programme leaders).

It needs an admin system so administrators can set up staff and module records and create programmes as collections of modules.
